# Raj Pithava - Personal Portfolio

Welcome to the repository for my personal portfolio! This is a modern, responsive, and highly interactive web application built to showcase my skills, projects, and professional background as a Full Stack Developer.

## 🚀 Live Demo
*(You can add your Vercel link here after deployment, e.g., `https://raj-portfolio.vercel.app`)*

## 🛠️ Built With
This portfolio was built using modern web technologies focusing on performance and elegant design:
- **[React 19](https://react.dev/)** - Component-based UI architecture
- **[Vite](https://vitejs.dev/)** - Next-generation frontend tooling for blazing fast builds
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first CSS framework for styling
- **[Lucide React](https://lucide.dev/)** - Clean and consistent iconography

## ✨ Key Features
- **Responsive Design**: Flawless experience across mobile, tablet, and desktop.
- **Glassmorphism UI**: High-end translucent design elements.
- **Advanced Animations**: Horizontal scroll-jacking for the Projects section and smooth reveal animations.
- **Dark/Light Mode**: Full support for system-preference or user-toggled theme switching.
- **Integrated AI Chatbot**: A custom TF-IDF NLP-powered chatbot running locally in the browser to answer questions about my resume and experience.
- **WhatsApp Integration**: Direct contact method with obfuscated credentials to prevent spam.

## 💻 Running Locally

If you want to run this project on your local machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/rajpanchal135/raj-portfolio.git
   cd raj-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
