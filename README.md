# 🚀 Raj Pithava - Personal Portfolio

Welcome to the repository for my personal portfolio! This is a modern, responsive, and interactive web application built to showcase my skills, projects, and professional background as a **Full Stack Developer**. The portfolio highlights my technical expertise, featured projects, achievements, and provides an easy way for recruiters and collaborators to connect with me.

---

## 🌐 Live Demo

**🔗 Portfolio Website:**
**https://raj-portfolio.vercel.app**

> Explore my portfolio to learn more about my projects, technical skills, experience, and contact information.

## 🛠️ Built With

This portfolio is built using modern web technologies with a focus on performance, scalability, and user experience.

* ⚛️ **React 19** – Component-based UI development
* ⚡ **Vite** – Lightning-fast frontend build tool
* 🎨 **Tailwind CSS v4** – Utility-first CSS framework
* 🎯 **Lucide React** – Modern and consistent icon library

---

## ✨ Features

* 📱 Fully Responsive Design for mobile, tablet, and desktop devices
* 🌟 Modern Glassmorphism UI
* 🎬 Smooth animations and interactive scrolling effects
* 🌙 Dark & Light Theme support
* 🤖 AI-powered Resume Chatbot (TF-IDF NLP based)
* 💼 Professional Projects Showcase
* 📄 Resume Download
* 📬 WhatsApp & Email Contact Integration

## 💻 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/rajpanchal135/Raj-Portfolio.git
cd Raj-Portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

### 5. Preview the production build

```bash
npm run preview
```

Open your browser and visit:

```
http://localhost:5173
```
